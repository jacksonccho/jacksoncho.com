Loading site for local changes:
1) type "yarn start"

Installing program
1) yarn install

Editing website
1) yarn start
2) site-text.json is where the text is located/stored
3) ctrl-s is save after an edit
4) ctrl shift f is global find

once changes are made-push to github
0) type "yarn build"
1) go to terminal and type "git add ."
2) type "git commit -m "message""
3) type "git push"
